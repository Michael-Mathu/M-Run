# Walk-Run Method

Start with a 1:1 walk:run ratio. After 2-3 weeks, shift to 2:1. Gradually increase running time over 8-12 weeks until you can run continuously.

**Week 1-2:** Run 1 min, walk 1 min
**Week 3-4:** Run 2 min, walk 1 min
**Week 5-6:** Run 3 min, walk 1 min
**Week 7+:** Run 5+ min, walk 1 min
