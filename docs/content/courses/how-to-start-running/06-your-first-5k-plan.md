# Your First 5K Plan

8-week training plan to get you to 5K:

**Weeks 1-2:** Run 1-2 min, walk 1 min (20-30 min total)
**Weeks 3-4:** Run 3-5 min, walk 1 min (25-35 min total)
**Weeks 5-6:** Run 5-8 min, walk 1 min (30-40 min total)
**Weeks 7-8:** Run 10-15 min, walk 1 min (35-45 min total)

Rest 1-2 days per week. Listen to your body.
